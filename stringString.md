문제:
프로그래머스 - 문자열안에 문자열

#### 코드

```
function solution(str1, str2) {
    var answer = 0;
    if(str1.includes(str2)){
        answer = 1;
    } else {
        answer = 2;
    }
    return answer;
}
```

includes 함수는 원래 배열에 특정 요소가 포함되어 있는지에 따라 true, false값을 반환하는 함수다
하지만 문자열에 쓰였을 때는 대소문자를 구분해 포함여부를 판단한다

위 코드에선 str1에 str2가 포함되어 있는지 확인, 포함되면 1, 아니면 2를 출력한다
